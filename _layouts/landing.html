Akumuli Timeseries Database

Akumuli is an open source Time Series Database available under Apache 2.0 License. The key goals of the project are efficiency and ability to work in stand-alone mode with little or no maintenance.

The database can scale up by utilizing multiple cores or CPUs. It's also can scale down and work on small and cheap VMs or in a "sidecar" container.

Checkout Akumuli on Github

Features

Storage

Column-oriented. SSD/NVMe optimized. Crash safety and recovery. State of the art compression, ~1GB/s encoding throughput.

Performance

Up to ~14 million writes per second on server. Up to ~1 million writes per second on cheap VM. Low latency reads.

Features

Fast aggregation, downsampling, and filtering without pre-computed rollups or materialized views. Hard limit on disk use. Can be used as an embedded library.

Integrations

Grafana plugin. Collectors via the OpenTSDB protocol.
